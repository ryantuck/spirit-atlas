## Structure

Nine original albums. 176 songs.

Structure is 2 group rounds followed by a bracket round.

Teardrops on My Guitar is the only song with two versions included in the discography. Pick the pop or radio single version to include.

    176 - 1 = 175

Randomly shuffle songs. Create playlist on Spotify as needed. Already-ordered is way easier to listen through, lesson from rap bracket.

Group into 35 groups of 5 songs.

    175 = 35 x 5

Rank each group. Top three advance.

    105 = 35 x 3

Group into 21 groups of 5 songs.

    105 = 21 x 5

Rank each new group. Top three advance.

    63 = 21 x 3

Pick one song that's been eliminated to bring it back in as the last entry in the tournament bracket <333.

    64 = 63 + 1
