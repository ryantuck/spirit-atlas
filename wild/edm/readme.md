# EDM

2500+ EDM tracks. Battle for mainstage.

## Structure

Pre-work - clean up 2800+ songs, remove duplicates, remove tracks not categorized properly.

Fetch core playlist of 2547 songs.

Shuffle.

Initial round - 512 groups of 5 songs, ranked 1-5. A couple groups have 4 songs each. 5th place eliminated. 1-4 seeded into next group round.

2048 songs in standard groups of 4. 1024 songs. 512 songs. 256 songs. 128 songs. Bracket.
